Welcome to Taskflow API

This API allows you to manage tasks. Below are the available endpoints:

Endpoints

  • POST /tasks - Add a new task
  • GET /tasks - Retrieve all tasks for the logged-in user
  • PUT /tasks/:id - Update task details (title, description, category)
  • DELETE /tasks/:id - Delete a task

Task Object

{
  "title": "Task Title",

  "category": "To-Do",
  "userId": "user123"
}

Authentication

Only authenticated users can access the API. Use Firebase Authentication (Google sign-in) to authenticate users.

Real-time Updates

The API supports real-time updates using WebSockets. Ensure that tasks remain in their last known order when the user refreshes or reopens the app.
